DOSSIÊ_TÉCNICO

O que faz OléFUT diferente.

F_01 / ENGINE

Motor Determinístico

Engine JavaScript ES2022 puro, zero-UI, headless. Mesma seed → mesma temporada, reprodutível. Simulação completa de 38 jogadas roda em <2 segundos. Cada gol, cartão e contusão tem causa traceable. Sem RNG black box.

F_02 / SCOPE

170 Clubes Reais

Brasileirão A-D. Premier League. La Liga. Serie A. Bundesliga. Ligue 1. Argentino. Estaduais brasileiros (Paulistão, Carioca, Mineiro, Gaúcho).

F_03 / OFFLINE

100% Offline

Roda 100% no navegador. Save em localStorage. Sem servidor, sem analytics, sem cloud. Privacidade absoluta.

F_04 / NARRATIVA

5 Camadas de Mito

Agente → Eventual → Relacional → Narrativa → Mito. Lendas geram traits herdáveis. Filhos regen reaparecem 18 anos depois. Rivalidades emergem do gameplay.

F_05 / STACK

React 19.2 + Vite 8

Stack moderna. Code-split agressivo: 376KB initial bundle gzipped 110KB. Tone.js opt-in pra synth audio chiptune.

F_06 / AKITA

Spec-Driven Development

144 SPECs versionadas. Cada feature: spec → harness → fix. Bug = ticket + fix + regression test. 1625 testes verdes no CI a cada PR. Sem vibe coding. Filosofia Fábio Akita aplicada sem exceção.

F_07 / BRAND

Premium 32-bit

Press Start 2P + Pixelify Sans + IBM Plex Mono. ISSSD-inspired. CRT scanlines. Sem pixel art barato.

F_08 / SOM

Audio Procedural

Sintetizado em runtime via Tone.js. Chiptune 8-canais. Click, gol, vitória, derrota — cada SFX gerado.

F_09 / LIVRE

Open Source

Repo público GitHub. Toda decisão arquitetural em CLAUDE.md. CHANGELOG honesto. Build in public.
