## 8. Visual Direction

### "Luxury Arcade 32-bit" Aesthetic

| Principle | Rule |
|-----------|------|
| Hard Edges | Zero blur, zero soft shadows. Pixel-perfect borders always. |
| Bevel & Emboss | Panels simulate depth via highlight top-left + shadow bottom-right. |
| Dithering over Gradients | Color transitions via dither pattern, never CSS gradients. |
| CRT Atmosphere | Dark backgrounds with subtle scanline. Phosphor green glow on highlights. |
| Collectible Card Frame | Shields/trophies/icons framed with metallic bevel + gold rivets. |
